# Finance Flow

Applicazione web/mobile per gestione patrimonio, spese, obiettivi e coaching finanziario.

## Requisiti

- Node.js `>= 22.0.0` (raccomandato: `22.22.0`)
- npm `>= 10`

## Setup locale

```sh
nvm install 22.22.0
nvm use
npm i
npm run dev
```

## Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- Supabase
